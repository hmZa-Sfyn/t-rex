# T-Rex Examples

This folder contains example scripts demonstrating module pipelines and error messages.

Files:
- `ip_sum.trex` — fetch IP, select `ip`, split by `.` and sum the parts.
- `loop_sum.trex` — generate a sequence with `forloop` and sum it.
- `if_example.trex` — evaluate a condition using `ifpy`.
- `error_example.trex` — runs `brokenmodule` to demonstrate error handling.

Usage:

Run a script with:

```bash
./t-rex examples/ip_sum.trex
```

Or start the shell and run the commands inside.
